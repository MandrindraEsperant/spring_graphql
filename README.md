# Spring GraphQL API

API GraphQL développée avec Spring Boot — CRUD pour les entités **Book** et **Author**.

## Technologies

- Java 17+
- Spring Boot
- Spring GraphQL
- Spring Data JPA
- PostgreSQL
- Maven

## Lancer le projet

```bash
./mvnw spring-boot:run
```

## Endpoint GraphQL
[text](http://localhost:8080/graphql)
